### **Class Object OOP Basic Concept**

OOP is about organizing code into objects that contain:
- Properties (Data)
- Method (Behavior)

### **Class & Object Cont.**
- Classes is the blueprint.
- Objects is the production of the blueprint.

##
### **OOP Main Principles:**
**Encapsulation:** hide data (private), expose only what's necessary.

**Inheritance:** sub classes get properties from a base class.

**Polymorphism:** same method, different behaviors (class).

**Abstraction:** hide implementation (only expose behavior)
